# Estudio de Realizaciones

Este es el proyecto final para el curso de Desarrollo Web en Coderhouse.

## 🗺️ Cómo navegar por el sitio
Para visualizar el proyecto de manera local, se debe iniciar desde el archivo principal:
1. Abrir el archivo index.html en la raíz del proyecto.
2. Utilizar el menú del encabezado para navegar de forma fluida entre las diferentes secciones.

## 🛠️ Correcciones implementadas
- Optimización de rutas relativas para la navegación entre directorios.
- Inclusión de etiquetas semánticas (<article>, <figure>, <footer>).
- Jerarquización de contenidos con etiquetas de títulos (<h1>, <h2>, <h3>).
